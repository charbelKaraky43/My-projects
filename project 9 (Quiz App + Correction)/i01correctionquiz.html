Question 1: Which language runs in a web browser?

a: Java
b: C
c: Python
d: JavaScript

The correct answer is : d

Explanation: A webpage can contain embedded JavaScript, which executes when a user visits the page. The language was created to allow web developers to embed executable code on their webpages, so that they could make their webpages interactive, or perform simple tasks.

Final question : How do you display hyperlinks without an underline?

a: a {underline:none;}
b: a {text-decoration:no-underline;}
c: a {text-decoration:none;}
d: a {decoration:no-underline;}

The correct answer is : c

Explanation: To remove the underline from a hyperlink, you need to use the CSS text-decoration property. Specifically, you need to use: text-decoration: none; . This tells the browser not to decorate the text (i.e. not to display an underline) .
